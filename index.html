🔥 Configurar Firebase
Configure uma vez e seus dados sincronizam em qualquer dispositivo — PC, celular, tablet.
1
Acesse console.firebase.google.com → selecione seu projeto
2
Build → Realtime Database → Criar banco → Modo de teste
3
Authentication → Métodos de login → Anônimo → Ativar
4
Project Settings → Seus apps → App Web → Copie a configuração
⚠️ Use o Realtime Database (não o Firestore). A databaseURL deve terminar em .firebaseio.com.
